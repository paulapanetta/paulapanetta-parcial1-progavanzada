Tecnologías

Java 17+
IntelliJ IDEA
Swing (JOptionPane) para la interfaz gráfica
Git + GitHub para control de versions

Cómo ejecutar

Clonar el repositorio:

bash   
git clone https://github.com/tu-usuario/GarageSystem.git

Abrir el proyecto en IntelliJ IDEA.
Ejecutar la clase Main.java.
Se abrirá una ventana solicitando la capacidad máxima del garage.
Navegar el menú interactivo con los botones de las ventanas emergentes.

Funcionalidades
Registrar ingreso
Alta de vehículo con validación de espacio y patente única
Registrar salida
Baja de vehículo con resumen de costo final
Listar vehículos
Muestra todos los vehículos estacionados actualmente
Estado del garage
Capacidad total, espacio ocupado y disponible
Reportes
Cantidad por tipo, recaudación estimada total
Salir
Cierra la aplicación
